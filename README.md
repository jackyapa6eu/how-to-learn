# **Научиться учиться**

Статичный сайт о том, как научиться учиться.
Приведены различные методики и советы, в том числе и от известных людей.

В проекте использовал такие технологии как:
* Flex.
* Grid-layout.
* Абсолютное позиционирование.
* Анимации.
* Псевдо-элементы.
* iframe
